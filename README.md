# PantryPal

## Problem
People often forget what grocery items they already have at home, leading to:
- Buying duplicate items they don't need
- Wasting food that expires before being used
- Forgetting essential items they actually need

## Solution
PantryPal is a simple, intuitive app that helps you:
- Track what items you currently have in your pantry
- Maintain a shopping list of items you need to buy
- Easily move items between your pantry and shopping list

## MVP Features
- **Pantry Tracking**: See all items currently in your pantry with quantities
- **Shopping List**: Maintain a list of items you need to purchase
- **Item Movement**: Easily move items between pantry and shopping list
- **Local Storage**: All data persists between sessions using browser localStorage

## Technology
- Vanilla JavaScript (no frameworks)
- HTML5/CSS3 for the interface
- localStorage for data persistence

## Demo Instructions
1. Open `index.html` in any modern web browser
2. Add items to either your pantry or shopping list
3. Move items between lists as needed
4. Refresh the page to see data persistence in action
